Exercise 1 :: 2025-04-22 :: Vite Dev Server with Questions and Answers

What is the role of the index.html file?

Ans : Website homepage.

What is the URL you used to view the page? Is it HTTP or HTTPS?

Ans : Look at the URL. If URL starts with https://, it's HTTPS but if URL starts with http://, it's HTTP.

What HTTP status code do you see for index.html? What does it mean?

Ans : 1xx: Informational , 2xx: Success (200 OK, 201 Created) , 3xx: Redirection (301, 302) , 4xx: Client Errors (404, 403) , 5xx: Server Errors (500, 503)

How can you tell the page is being served locally vs from a real domain?

Ans : Look at the URL. If URL contains localhost, 127.0.0.1, or file:///, its hosted locally but if URL contains something like https://domainname.com, it's hosted on a real domain.

What tools did you use to inspect the response?

Ans : Use Developer Tools.
